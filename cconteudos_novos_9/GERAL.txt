===== CADASTRO DE GRUPO =====

Como cadastrar grupos?

Para realizar o cadastro do grupo basta seguir o seguinte caminho:
Cadastro > Usuário e Grupo > Grupo > Incluir



O cadastro do grupo serve para identificar as telas que cada usuário terá acesso.
Ex: Podemos criar um grupo com o nome "Operação" e nele selecionar todas as telas que os usuários que fazem parte da operação, poderão utilizar. 

Campo 1 - Nome do Grupo: É a identificação do grupo que será criado;
Campo 2 - Perfil de acesso: São as telas que os usuários pertencentes a este grupo poderão ter acesso.

Obs: Os botões "Expandir" e "Contrair" servem para ajustar o modo de exibição das opções do perfil de acesso. 



Após selecionar as telas, clicamos em salvar e o grupo criado já estará disponível para ser selecionado no cadastro de usuário. O mesmo poderá ser editado/atualizado a qualquer momento pelos usuários que fazem parte do grupo "administrador", basta seguir o seguinte caminho e clicar duas vezes no grupo que deseja realizar o ajuste:
Cadastro > Usuário e Grupo > Grupo > Consultar


===== CADASTRO DE USUÁRIO =====

Como cadastrar usuários?

Para realizar o cadastro do usuário basta seguir o seguinte caminho:
Cadastro > Usuário e Grupo > Usuário > Incluir



Aba Informações:
*Foto Padrão: É opcional inserir ou não uma foto do usuário.
Campo 1 - Nome: Nome do usuário que será cadastrado;
Campo 2 - Telefone: Número de Telefone Fixo do usuário (caso não tenha, apenas preencher com "11");
Campo 3 - Celular: Número de Celular do usuário;
Campo 4 - CPF: Número do CPF do usuário;
Campo 5 - E-mail: E-mail corporativo do usuário, onde será enviado a senha para o usuário realizar login no TMS;
Campo 6 - Data de Nascimento: Data de nascimento do usuário;
Campo 7 - Sexo: Gênero do usuário;
Campo 8 - Obs: Pode ser o cargo, setor ou qualquer outra informação considerada válida;
Campo 9 - Resetar Senha: Caso o usuário esqueça a senha, a mesma poderá ser resetada a qualquer momento.



Aba Segurança:
Nessa aba é necessário preencher a qual grupo o usuário terá acesso (basicamente quais informações ele poderá ver), isso pode ser informações da operação, administrador, clientes... E os níveis e assuntos de chamados que eles poderão abrir internamente quando estiver com algum problema/dúvida.
Campo 10 - Grupo de Acesso: Qual grupo o usuário terá acesso (basicamente quais informações ele poderá ver), isso pode ser informações da operação, administrador, clientes...
Campo 11 - Grupo de Indenização: É pra quando o cliente contrata esse módulo e são criados grupo de indenização que são basicamente, grupos de atendimento.
Campo 12 - Nível de chamado: Qual o nível de chamado interno ele poderá abrir, isso vai depender dos níveis que a transportadora cadastrar;
Campo 13 - Assunto: Qual o assunto do chamado, isso também deverá ser cadastrado pela transportadora.



Aba Endereço:
Nessa aba deverá ser preenchido as informações do endereço do próprio usuário.
Campo 14 - CEP: CEP da residência do usuário (ao preencher o CEP, as informações: logradouro, bairro, estado e município, são preenchidas automaticamente pelo sistema);
Campo 15 - Número: Número da residência do usuário;
Campo 16 - Complemento: Complemento da residência do usuário (não é obrigatório inserir).



Aba Logística:
Essa aba não é de preenchimento obrigatório, mas serve para identificarmos qual cliente este usuário atende (caso o usuário faça atendimento para um cliente especifico). 
Campo 17 - Remetente: Qual cliente da transportadora que este usuário atende;
Campo 18 - Filial Remetente: Quais filiais deste cliente o usuário atende;
Campo 19 - Transportador: Qual transportadora este usuário atende;
Campo 20 - Filial transportador: Quais filiais especificas da transportadora este usuário atende;
Campo 21 - Unidade: Qual base da transportadora o cliente atende;



Após realizar o preenchimento de todos os campos, clique em salvar.

===== CADASTRO DE ROTA TRANSFERÊNCIA =====

Como cadastrar rota transferências?

Para realizar cadastrar uma rota de transferência para outra filial/base basta seguir o seguinte caminho:
Cadastro > Operacional > Rota Transferência > Incluir





Nessa tela é possível realizar o cadastro de rotas de transferência entre filiais/bases

Campo 1 - Nome da rota: Nomenclatura  da rota que será criada;
Campo 2 - Código da rota: Código interno da rota (Pode ser ultilizado as UFs de destino, exemplo: SPMG);
Campo 3 - Percurso Vinculado: Caso o percurso da rota não esteja criado, é necessário adicionar o percurso da rota (Tela: Cadastro > Operacional > Percurso > Incluir);
Campo 4 - Transportador Filial;
Campo 5 - Filial de destino: Filial de destino da transferência;
Campo 6 - Filial de origem: Filial de origem da transferência;       
Campo 7 - Selecione os transbordos: Caso tenha algum transbordo durante o trajeto, basta selecionar a rota e arrastar até o campo "Transbordos Selecionados" 
Campo 8 - Cancelar: Caso queira cancelar o cadastro aperte "Cancelar";
Campo 9 - Salvar: Caso os campos estverem preenchidos corretamente aperte "Salvar"



Após salvar a rota de transferência entre filiais/bases será criada;

===== CADASTRO DE CLIENTE SERVIÇO =====

Como cadastrar cliente serviços?

Para realizar o cadastro de Cliente Serviço basta seguir o seguinte caminho:
Cadastro > Serviço > Cliente Serviço > Incluir 



Nessa tela cadastramos o tipo de serviço que determinado remetente/cliente do meu cliente irá utilizar. 

Campo 1 - Remetente: É onde inserimos o cliente que foi criado;
Campo 2 - Serviço: O tipo de serviço que este cliente vai utilizar;
Campo 3 - Tabela - Preço: A tabela de preço que este cliente vai usar;
Campo 4 - Cálculo de frete: O tipo de cálculo, se é calculado por remessa, volume ou lote;
Campo 5 - CTE: Informar se o CTE será emitido por remessa, volume, lote rateado, lote CTE único ou nenhum;
Campo 6 - Emite Nota Fiscal: Se o cliente emite ou não nota fiscal;
Campo 7 - Serviço CTE: Qual o tipo de serviço do CT-e, se é normal, subcontratação, redespacho, ou outros;
Campo 8 - Gris - %: Se possui gris. O GRIS é uma taxa cobrada no valor da nota fiscal, em forma de percentual. Ela existe com o objetivo de cobrir os custos ligados ao combate contra roubo de cargas durante o transporte;
Campo 9 - Ad Valorem - %: Se possui ad valorem. É uma taxa cobrada dentro da tabela de fretes que representa o custo do seguro da carga. Seu cálculo é feito através da porcentagem da mercadoria sobre o valor da Nota Fiscal da carga transportada agregada ao custo de frete;
Campo 10 - Fator Cubagem: É um valor que equivale ao peso ideal para cada metro cúbico do interior do veículo;
Campo 11 - Peso - Cubagem Mínima: O espaço mínimo que a carga irá realmente ocupar no veículo;
Campo 12 - Cobra Mecanizado: A mecanização de certos processos serve para empresas que realizam a movimentação padronizada de certos materiais;
Campo 13 - Medida do Mecanizado: Qual a medida do mecanizado;
Campo 14 - Valor de Taxa de Mecanizado: Quanto este cliente cobra de taxa de mecanizado;
Campo 15 - Método de Cálculo: Se vai calcular por percentual da nota fiscal, por kg, etc;
Campo 16 - Valor de Averbação Mínima: A averbação de carga é o ato da transportadora comunicar à seguradora quais produtos serão transportados no frete e as demais informações da operação, a fim de que seguro de carga seja autorizado. Qual é o valor mínimo que este cliente cobra de averbação;
Campo 17 - Valor de Averbação Máxima: Qual é o valor máximo que este cliente cobra de averbação;
Campo 18 - Tipo Gatilho CTE: Por qual gatilho o sistema vai emitir o CTE (se vai ser no recebimento, no processamento, no carregamento, na importação);
Campo 19 - TP Faturamento Importação:
Campo 20 - Tipo Reentrega: Se é fixo ou percentual;
Campo 21 - Taxa Reentrega: Valor da taxa de reentrega;
Campo 22 - Tipo Devolução: Se é fixo ou percentual;
Campo 23 - Taxa de Devolução: Valor da taxa de devolução.


Após o preenchimento de todos os campos, basta clicar em salvar ao final da tela e o seu cliente serviço estará criado.


===== ERRO DE VALOR DO ICMS =====

Como corrigir erro de valor do ICMS no CTE?

1° Passo: Clique 2 vezes em "Rejeitado" para entrar na tela de correção;



2° Passo: Vá para a aba "CTE" e copie o valor, após isso apague os campos "Valor do CTE" e "Valor do ICMS"



3° Passo: Clique no botão "Alterar valor CTE". Depois disso, adicione o valor copiado anteriormente e clique em "Gravar".



Após os passos acima, o valor do ICMS será calculado automaticamente, basta salvar a correção e reenviar o CTE.



===== CADASTRO DE FILIAL GRUPO EMPRESARIAL =====

Como cadastrar filial grupo empresariais?

Em certos casos com nossos clientes, os motoristas realizam todas as baixas através do aplicativo, porém as listas não finalizam, isso ocorre por conta das "Baixas Automáticas" não estarem ativadas.

Para ativar, siga o caminho abaixo:
Cadastro > Grupo Empresarial > Filial Grupo Empresarial > Consultar



Link para acessar o tutorial sobre a tela:
https://tmslog.com.br/Adm_Tutoriais.aspx?id=32

1° Passo - Busque a filial desejada, após isso dê 2 cliques em cima da mesma;



2° Passo - Procure o campo "Baixa Automática" e ative-o;


3° Passo - É importante verificar os horários que são permitidos no campo "Faixa de entrega" este campo trata-se basicamente do
horário que é permitido realizar as baixas, tanto no aplicativo como no sistema;





Após a ativação do campo, as próximas listas com pedidos com ocorrência finalizadora, serão finalizadas automaticamente.



===== CADASTRO DE SEGUROS =====

Como cadastrar seguro(s)?

Para cadastrar seguro(s) basta seguir o seguinte caminho:
Cadastro > Seguros > Cadastrar



Nessa tela, realizamos o cadastro do(s) seguro(s).

Campo 1 - Seguradora: Nome da seguradora;
Campo 2 - CNPJ;
Campo 3 - N° da Apólice: Número da apólice do seguro;
Campo 4 - Usuário: Login (Pode ser ficitício);
Campo 5 - Senha: (Pode ser ficitício);
Campo 6 - Código ATM: Código responsável pela averbação automatíca dos documentos fiscais de transporte;
Campo 7 - Seguro Ramo: Ramo da seguradora;
Campo 8 - Padrão: Seguro padrão ou não;



Após o preenchimento dos campos, basta salvar.

Obs: Caso aconteça algum problema nos envios dos documentos fiscais de transporte, entre em contato.
