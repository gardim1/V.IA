===== RECEBIMENTO TRANSFERÊNCIA COM FALTANTES =====

O caminho para acessar "RECEBIMENTO TRANSFERÊNCIA COM FALTANTES" é: Operação > Recebimento Transferência com Faltantes

Como realizar recebimento transferência com faltantes:


Para realizar o Recebimento transferência com faltantes na filial de descarregamento, basta seguir o seguinte caminho:
Operação > Recebimento Transferência com Faltantes



Campo 1 - Carregamento: Bipe o número da carga e dê "Enter";
Campo 2 - Identificador: Bipe o número da consolidação.


Após bipar todas as consolidações da carga, caso haja alguma falta, selecione o botão 'i' e receba com faltantes. Logo após a transferência será recebida e o próximo passo é fazer a desconsolidação dos pedidos.

===== RECEBIMENTO HUB =====

O caminho para acessar "RECEBIMENTO HUB" é: Operação > Recebimento Hub

Como realizar recebimento hub:


Para realizar o Recebimento de um pedido na base basta seguir o seguinte caminho:
Operação > Recebimento Hub

 

Nessa tela realizamos a primeira etapa da operação, que é o recebimento do pedido na base, essa etapa serve para conferir se os pacotes foram entregues corretamente.

Campo 1 - Remetente Filial: Selecionamos a filial do cliente que se refere aquele pedido;
Campo 2 - Código de Barras: É o código que foi informado que seria bipado. Esse código é passado na integração, onde perguntamos qual código será bipado na operação podendo ser: Chave ou número da nota, número do pedido ou qualquer outro que o cliente deseja bipar.  Esse código é bipado ou pode ser digitado.


Após realizar este procedimento, o pedido já estará com o status de recebido e deverá seguir para o processamento.

===== RECEBIMENTO TRANSFERÊNCIA =====

O caminho para acessar "RECEBIMENTO TRANSFERÊNCIA" é: Operação > Recebimento Transferência

Como realizar recebimento transferência:


Para realizar o Recebimento dos pedidos na filial/base de descarregamento, basta seguir o seguinte caminho:
Operação > Recebimento Transferência



Nessa tela é realizado o recebimento do pedido que foi transferido para a filial/base em questão (filial de destino). 

Campo 1 - Carregamento: Bipe apenas o número da carga e dê "Enter";



Após dar enter, o segundo campo estará disponível para preencher e o sistema irá mostrar a quantidade de consolidações que há dentro da carga e que devem ser bipadas (campos em vermelho):

Campo 2 - Identificador: Bipe o número da consolidação.



Após bipar todas as consolidações da carga, o sistema irá identificar automaticamente a ultima consolidação bipada, gerando no topo da tela uma mensagem de sucesso e o número da lista. Os pedidos já estarão com o status de Transferido e deverá seguir para a desconsolidação.

===== RECEBIMENTO LISTA ENTREGA =====

O caminho para acessar "RECEBIMENTO LISTA ENTREGA" é: Operação > Recebimento Lista Entrega

Como realizar recebimento lista entrega:


Para realizar o Recebimento da Lista de entrega, basta seguir o seguinte caminho:
Operação > Recebimento Lista Entrega



Nessa tela realizamos o recebimento da lista de entrega. 

Campo 1 - Número da Lista: Bipe o número da lista de entrega que foi criada e dê "Enter";



Ao dar enter, o segundo campo ficará disponível para preencher e o sistema irá mostrar a quantidade de pedidos que há dentro da lista e que não foram entregues pelo motorista (campos em vermelho):

Campo 2 - Identificador: Bipe todos os códigos dos volumes que não foram entregues dessa lista. 



Ao bipar todos os volumes da lista, o sistema irá identificar automaticamente o ultimo pedido bipado, gerando no topo da tela uma mensagem de sucesso, e automaticamente o pedido estará liberado para ser colocado em rota novamente.

===== ETIQUETA RECEBIMENTO =====

O caminho para acessar "ETIQUETA RECEBIMENTO" é: Operação > Etiqueta > Etiqueta Recebimento

Como realizar etiqueta recebimento:


Para realizar a impressão da Etiqueta de Recebimento basta seguir o seguinte caminho:
Operação > Etiqueta > Etiqueta Recebimento



Nessa tela realizamos a impressão de etiquetas de recebimento.

Campo 1 - Data de Emissão: Data de emissão das etiquetas que deseja pesquisar (data inicial);
Campo 2 - Até: Até qual data deseja pesquisar (data final);
Campo 3 - Impressora: Selecione a impressora que fará a impressão da etiqueta;
Campo 4 - Nº Volume da Remessa: Número de volume da remessa que deseja imprimir a etiqueta;
Campo 5 - Nº Remessa: Número da remessa que deseja imprimir a etiqueta;

Obs: Não é obrigatório o preenchimento de todos os filtros, os mesmos estão na tela para auxiliar o usuário a realizar as buscas. Após indicar o filtro, clique em pesquisar.



Campo 6 - Remessas: Pesquise uma remessa especifica que deseja imprimir;
Campo 7 - Selecionar: Selecione as etiquetas e indique algum dos botões no canto superior da tela;
Botão Ocultar/Exibir Paginação: Botão Ocultar/Exibir Paginação: Oculta o número de páginas e lista a etiquetas, ou exibe as numerações das páginas;

Botão Alternar: Alterna o modo de exibição da página;

Botão Tela Cheia: Exibe as etiquetas em tela cheia;

Botão Colunas: Exibe apenas as colunas desejadas pelo usuário;

Botão Classificação Múltipla: Exibe o relatório com uma classificação desejada, pode ser inserido uma coluna e a ordem que deseja visualizar a mesma (Ascendente ou Descendente);

Botão Exportar Dados (CSV, TXT, MS-Excel, (Open XML) e PNG): Baixa as etiquetas com o formato de arquivo desejado;

Botão Imprimir Etiqueta dos Selecionados: Realiza a impressão de etiquetas selecionadas.

===== RELATÓRIO FALTANTES RECEBIMENTO =====

O caminho para acessar "RELATÓRIO FALTANTES RECEBIMENTO" é: Operação > Relatório Operação > Relatório Faltantes Recebimento

Como realizar relatório faltantes recebimento:


Para verificar o Relatório Faltantes Recebimento basta seguir o seguinte caminho:
Operação > Relatório Operação > Relatório Faltantes Recebimento



Nessa tela verificamos os pedidos que faltam ser recebidos na base. 

Campo 1 - Data de Início: Serve para identificar a partir de qual data que será verificado (data inicial);
Campo 2 - Até: Até qual data será verificado (data final);
Campo 3 - Nº da Remessa: Número da remessa que deseja pesquisar;
Campo 4 - Nº Volume da Remessa: Indique o volume que deseja pesquisar, caso queira puxar por um pedido específico;
Campo 5 - Código de Barras: Bipar novamente o código que foi informado no ato da configuração do cliente;
Campo 6 - Remetente: Indique a base/filial que enviou os pedidos;
Campo 7 - Pesquisar: Clique para pesquisar através dos filtros selecionados;
Campo 8 - Download: Clique em download para baixar o relatório em excel;

Obs: Não é obrigatório o preenchimento de todos os filtros, os filtros estão na tela para auxiliar o usuário a realizar as buscas.
